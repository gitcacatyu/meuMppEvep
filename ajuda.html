Ajuda População

Se você está com dificuldades ou tem dúvidas sobre o MPP, aqui estão algumas dicas:

MPP Console

O console é uma ferramenta simples, mas poderosa com comandos JavaScript.

Você pode executar comandos JavaScript diretamente no console.

Use o botão "Executar" para ver os resultados.

Para limpar o console, clique no botão "Limpar Console".

Configuração

Na página de configuração, você pode ajustar as preferências do MPP.

Use o botão "Configurar" para acessar as opções.

Mini-Jogos

Explore os mini-jogos disponíveis no MPP.

Use o botão "Jogos" para acessar a lista de jogos.

Instalação

Para instalar o MPP, siga as instruções na página de instalação.

Use o botão "Instalar" para iniciar o processo.

Sobre o Sistema

Na página "Sobre", você encontrará informações sobre o MPP e sua história.

Use o botão "Sobre" para acessar essas informações.


Horários de funcionamento/Ajuda

Feriados, Sábados e Domingos: 16:00 ás 19:30.
