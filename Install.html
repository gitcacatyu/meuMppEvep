Instalar MPP

Para instalar o MPP, siga as instruções abaixo:

  1. Baixe o arquivo de instalação do MPP.
  2. Abra o terminal e navegue até o diretório onde o arquivo foi baixado.
  3. Execute o comando: bash install.sh
  4. Siga as instruções na tela para completar a instalação.

Após a instalação, você pode iniciar o MPP digitando mpp no terminal.
